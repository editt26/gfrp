## GFRP คืออะไร

GFRP ย่อมาจาก Glass Fiber Reinforced Polymer เป็นวัสดุคอมโพสิตที่ใช้เส้นใยแก้วเป็นตัวเสริมแรงและใช้เรซินเป็นตัวประสาน เมื่อนำมาผลิตเป็นเส้นสำหรับงานคอนกรีตมักเรียกว่า GFRP Rebar หรือเหล็กเส้นไฟเบอร์กลาส

จุดเด่นของวัสดุชนิดนี้คือไม่เป็นสนิมจากตัววัสดุ น้ำหนักเบา และไม่นำไฟฟ้า จึงถูกพูดถึงมากในงานที่มีความชื้น ความเค็ม หรือสภาพแวดล้อมกัดกร่อน เช่น งานริมทะเล งานบ่อบำบัด งานพื้นคอนกรีตบางประเภท และงานอุตสาหกรรม

## GFRP Rebar ใช้กับงานอะไร

GFRP Rebar มักถูกพิจารณาในงานที่ต้องการลดความเสี่ยงจากสนิมหรือการกัดกร่อนของวัสดุเสริมแรงในคอนกรีต ตัวอย่างงานที่พบได้บ่อยคือ:

- งานพื้นคอนกรีตและลานคอนกรีต
- งานริมทะเล ท่าเรือ และพื้นที่ที่มีไอเกลือ
- งานบ่อบำบัดน้ำเสียหรือพื้นที่เปียกชื้น
- งานโรงงานหรือพื้นที่ที่มีสารกัดกร่อนบางประเภท
- งานเฉพาะทางที่ต้องการวัสดุไม่นำไฟฟ้า

การเลือกใช้ควรตรวจแบบและเงื่อนไขโครงสร้างกับวิศวกรก่อนเสมอ เพราะ GFRP ไม่ได้มีพฤติกรรมเหมือนเหล็กเส้นทุกด้าน

## ราคา GFRP ขึ้นกับอะไร

ราคา GFRP Rebar ไม่ควรดูเป็นตัวเลขเดียวโดยไม่เทียบสเปก เพราะราคาจริงขึ้นกับหลายปัจจัย:

- ขนาดเส้น GFRP
- ความยาวต่อเส้น
- ปริมาณรวมตาม BOQ
- ประเภทงานและข้อกำหนดโครงการ
- จังหวัดจัดส่งและเงื่อนไขขนส่ง
- รอบส่งมอบและความเร่งด่วน

ถ้ามี BOQ หรือแบบก่อสร้าง ควรเตรียมไว้ก่อนขอราคา เพราะจะช่วยให้ผู้ขายประเมินปริมาณและต้นทุนได้ใกล้เคียงงานจริงมากกว่า

## คำถามที่พบบ่อย

### GFRP Rebar คือเหล็กเส้นหรือไม่

GFRP Rebar เป็นวัสดุเสริมแรงคอนกรีตเหมือนเหล็กเส้นในแง่การใช้งานบางประเภท แต่ตัววัสดุไม่ใช่เหล็ก เพราะผลิตจากเส้นใยแก้วและเรซิน

### GFRP ใช้แทนเหล็กเส้นได้ทุกงานไหม

ไม่ควรใช้แทนทุกงานโดยอัตโนมัติ ต้องตรวจแบบ มาตรฐาน และคำแนะนำจากวิศวกรก่อนเลือกใช้

### ก่อนขอราคา GFRP ต้องเตรียมอะไร

ควรเตรียม BOQ ขนาดเส้น ปริมาณ ความยาวต่อเส้น จังหวัดจัดส่ง และข้อมูลประเภทงาน เพื่อให้ประเมินราคาได้แม่นยำขึ้น
